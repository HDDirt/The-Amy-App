Install on iPhone

  1. Clone the repository: git clone https://github.com/HDDirt/The-Amy-App.git
  2. Install dependencies: npm install
  3. Initialize iOS project: npm run ios:init && npm run ios:add
  4. Build web assets: npm run build
  5. Sync to iOS: npm run ios:sync
  6. Open in Xcode: npm run ios:open
  7. In Xcode:
    • Select your team for signing
    • Choose your device
    • Click Run (▶️)

For TestFlight distribution, see the Apple TestFlight documentation.
